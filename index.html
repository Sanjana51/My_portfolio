I developed a Retrieval-Augmented Generation (RAG) AI assistant that allows users to interact with PDFs via a chatbot and receive concise, context-aware summaries. The system integrates ChromaDB for efficient vector storage and retrieval, and uses Google's Gemini 1.5 model for high-quality question answering. I also built a user-friendly interface with HTML, CSS, and LangChain to support seamless document processing and intelligent search..

IMDB Sentiment-Analysis

I developed a Sentiment Analysis model using LSTM with TensorFlow and Keras to classify IMDb movie reviews as positive or negative. The model was trained on a dataset of 50,000 reviews, with preprocessing steps like tokenization and padding to standardize input lengths. An interactive Gradio interface was integrated to allow real-time sentiment predictions from user input..
